# Binary Number Operations Using Doubly Linked List

## Program Output Format

### Input
```
Enter first binary number: 10101
```

### Stored Binary Number
```
10101
```

### 1's Complement
```
01010
```

### 2's Complement
```
01011
```

### Second Binary Number Input
```
Enter second binary number: 1101
```

### Addition Result
```
100010
```
