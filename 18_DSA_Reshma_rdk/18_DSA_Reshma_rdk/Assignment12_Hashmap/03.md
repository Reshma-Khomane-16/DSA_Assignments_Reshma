
# Assignment — Employee Database Simulation Using Mid-Square Hashing (Student Generated)

## Title
Simulation of Employee Database Using Mid-Square Hash Function With Linear Probing Collision Handling

## Objective
To write a program that stores and searches employee (faculty) records using a hash table with:
- Mid-Square Hash Function  
- Linear Probing  
- Employee ID as the key  

## Theory

A hash table stores records using a hash function that maps a key to an index.  
In this assignment:
- **Key** = Employee ID  
- **Value** = Employee Name & Department  

### Mid-Square Hash Function
1. Square the key  
2. Extract middle digits  
3. Take modulo table size  

Example:
```
Key = 123
123² = 15129
Middle = 12
Index = 12 % size
```

### Linear Probing
If collision occurs:
```
newIndex = (index + 1) % size
```

---

## Algorithm

### Insert Employee
1. Compute hash using mid-square method  
2. If slot empty → insert  
3. Else linear probe until empty slot found  

### Search Employee
1. Compute index  
2. If match → return  
3. Else linear probe  
4. If found → display  
5. Else → not found

## Code (C++)
```cpp
#include <iostream>
#include <cmath>
#include <string>
using namespace std;

struct Faculty {
    int id;
    string name;
    string department;
};

const int SIZE = 10; // Hash table size
Faculty hashTable[SIZE];
bool occupied[SIZE] = {false};

// Mid-square hash function
int midSquareHash(int key) {
    int square = key * key;
    int digits = log10(square) + 1;
    int mid;
    if (digits <= 2) {
        mid = square % SIZE;
    } else {
        int shift = digits / 2;
        mid = (square / (int)pow(10, shift - 1)) % 100; // take middle 2 digits
        mid = mid % SIZE;
    }
    return mid;
}

// Insert faculty record
void insertFaculty() {
    Faculty f;
    cout << "Enter Faculty ID: ";
    cin >> f.id;
    cout << "Enter Faculty Name: ";
    cin.ignore();
    getline(cin, f.name);
    cout << "Enter Department: ";
    getline(cin, f.department);

    int index = midSquareHash(f.id);
    int originalIndex = index;
    
    // Linear probing
    while (occupied[index]) {
        index = (index + 1) % SIZE;
        if (index == originalIndex) {
            cout << "Hash table is full!\n";
            return;
        }
    }
    
    hashTable[index] = f;
    occupied[index] = true;
    cout << "Faculty inserted at index " << index << "\n";
}

// Search faculty by ID
void searchFaculty() {
    int id;
    cout << "Enter Faculty ID to search: ";
    cin >> id;
    int index = midSquareHash(id);
    int originalIndex = index;
    
    while (occupied[index]) {
        if (hashTable[index].id == id) {
            cout << "Faculty found at index " << index << ":\n";
            cout << "ID: " << hashTable[index].id << "\n";
            cout << "Name: " << hashTable[index].name << "\n";
            cout << "Department: " << hashTable[index].department << "\n";
            return;
        }
        index = (index + 1) % SIZE;
        if (index == originalIndex) break;
    }
    cout << "Faculty not found!\n";
}

// Display hash table
void displayTable() {
    cout << "Hash Table:\n";
    for (int i = 0; i < SIZE; i++) {
        cout << i << ": ";
        if (occupied[i])
            cout << hashTable[i].id << " - " << hashTable[i].name << " - " << hashTable[i].department;
        cout << "\n";
    }
}

int main() {
    int choice;
    do {
        cout << "\n--- Faculty Hash Table Menu ---\n";
        cout << "1. Insert Faculty\n";
        cout << "2. Search Faculty\n";
        cout << "3. Display Table\n";
        cout << "4. Exit\n";
        cout << "Enter your choice: ";
        cin >> choice;

        switch(choice) {
            case 1: insertFaculty(); break;
            case 2: searchFaculty(); break;
            case 3: displayTable(); break;
            case 4: cout << "Exiting...\n"; break;
            default: cout << "Invalid choice!\n";
        }
    } while(choice != 4);

    return 0;
}

```
